# Sales App

### Description

This is a simple full stack CRUD project made for node/graphql/apollo/postgreSQL practicing puprposes. User can register/login, add his own sales and products, update them and of course also delete.

### Technologies used

#### Front-end

* reactjs
* apollo-client
* material-ui
* formik
* graphql
* typescript

#### Back-end

* nodejs
* express
* graphql
* type-graphql
* typeorm
* apollo-server
* jsonwebtoken
* bcyptjs
* postgreSQL
* typescript
